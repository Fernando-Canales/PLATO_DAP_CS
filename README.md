# Detecting False Positives with Double-aperture photometry and Centroid Shifts

## Fernando Gutiérrez-Canales

### (under the supervision of Dr. Réza Samadi and Dr. Aaron Birch)

## Description

This is the main repository of my PhD project at LIRA (LESIA) and MPS. The files in this repository are devoted to my project about the detection of  false planetary transits (False Positives) within the on-board extraction photometry procedure of the [PLATO space mission](https://www.esa.int/Science_Exploration/Space_Science/Plato).

The scientific results obtained with the present code can be find in the following peer-reviewed publication: [Detecting False Positives with PLATO using double-aperture photometry and centroid shifts](https://doi.org/10.1051/0004-6361/202453294). These results were important for the correct implementation of false-positive prevention techniques for the PLATO space mission data pipeline.
